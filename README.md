# Framebuffer-Test

Compilation :
  gcc -o framebuffer-test framebuffer-test.c

Run in MacOS's Virtualbox:
  Press fn + control + option + F2 for switching to command line mode
  Run framebuffer-test
  
Run in CentOS:
  Press Ctrl + Alt + F2 for switching to command line mode
   Run framebuffer-test
   
   
Enjoy...!!
